## Special Files and Required Locations

- `.env`: place at project root (`/home/sabbu/Projects/college/.env`)
- `composer.json` / `composer.lock`: project root (run `composer install`)
- `vendor/`: generated by Composer at project root
- `.htaccess`: project root (Apache rewrite/security rules)
- `config/db.php`: database bootstrap (reads `.env`)
- `index.php`: primary entrypoint/home
- `pages/`: route pages (`/pages/bca`, `/pages/mca`, etc.)
- `BCA/`, `MCA/`, `IBM/`: main study-content roots consumed by explorer and viewer
- `uploads/`: upload destination (must be writable by web server)
- `output/allFiles/`: generated output destination for `output/output.php` (must be writable)
- `compiler/temp/`: temporary execution files for `compiler/run.php` (must be writable or creatable)
- `assets/components/`: shared layout/components (`navbar.php`, `footer.php`, `scripts.php`)
- `pages/search.php`: search API endpoint scanning server document root
- `pages/view.php`: controlled file preview endpoint (`?file=/BCA/...` etc.)
- `admin/`: admin partials included from `pages/admin.php`
- `dumbcli/install.sh` and `dumbcli/install.ps1`: intentionally exposed by `.htaccess` allow-list

## Minimum Local Run Checklist

1. Put `.env` in project root with `MODE`, `DB_HOST`, `DB_USER`, `DB_PASS`, `DB_NAME`.
2. Run `composer install` in project root so `vendor/` exists.
3. Ensure these folders are writable by PHP: `uploads/`, `output/allFiles/`, `compiler/temp/`.
4. Serve with Apache + `mod_rewrite` so `.htaccess` routes work.
5. Create DB tables used by code: `courses`, `feedback`, `ip_addresses`.
